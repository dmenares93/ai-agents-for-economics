# AI Agents for Economics

Notas y guías prácticas sobre el uso de **agentes de IA** en economía: cómo instalarlos, configurarlos y sacarles provecho en flujos reales de investigación, programación y análisis de datos con **Stata**.

Este repo es una colección de **temas independientes**. No es un curso lineal: cada carpeta es un tema autocontenido que puedes leer en el orden que quieras.

## Temas

- [intro-ai-agents/](intro-ai-agents/) — Introducción a los agentes de IA: qué es un LLM, qué es un agente, modelos estado del arte, IDEs donde correrlos, tips generales.
- [workflow/](workflow/) — Flujo de trabajo con **VS Code / Cursor**, Claude Code, Codex, extensiones útiles y configuración para investigación y análisis de datos en Stata.
- [prompt-engineering/](prompt-engineering/) — Técnicas de ingeniería de prompts: patrones, ejemplos, cuándo usar qué.

## Otros directorios

- [examples/](examples/) — Ejemplos de código y notebooks compartidos entre temas.

## A quién está dirigido

Economistas, investigadores y estudiantes dedicados a la investigación, programación y análisis de datos con Stata que quieren incorporar agentes de IA a su flujo de trabajo diario.
