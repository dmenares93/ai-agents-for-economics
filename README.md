# AI Agents for Economics

Notas y guías prácticas sobre el uso de **agentes de IA** en economía y programación: cómo instalarlos, configurarlos y sacarles provecho en flujos de trabajo reales (análisis de datos, Stata, Python, escritura, etc.).

Este repo es una colección de **temas independientes**. No es un curso lineal: cada carpeta es un tema autocontenido que puedes leer en el orden que quieras.

## Temas

- [intro-ai-agents/](intro-ai-agents/) — Introducción a los agentes de IA: qué es un LLM, qué es un agente, modelos estado del arte, IDEs donde correrlos, tips generales.
- [ide-setup/](ide-setup/) — Instalación y configuración de **VS Code / Cursor**, Claude Code, Codex, extensiones útiles y flujo de trabajo para economía/data.
- [prompt-engineering/](prompt-engineering/) — Técnicas de ingeniería de prompts: patrones, ejemplos, cuándo usar qué.

## Otros directorios

- [examples/](examples/) — Ejemplos de código y notebooks compartidos entre temas.

## A quién está dirigido

Economistas, investigadores y estudiantes que trabajan con datos (Stata, Python, R) y quieren incorporar agentes de IA a su flujo de trabajo diario.

## Contribuciones

Este repo crece con el tiempo. Si encuentras errores o tienes sugerencias, abre un issue o un PR.
